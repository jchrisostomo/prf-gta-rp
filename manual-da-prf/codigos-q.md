# Códigos Q

A comunicação via rádio utiliza os **Códigos Q**, que são fundamentais para o trabalho policial.

| Código | Descrição              |
| ------ | ---------------------- |
| QAP    | Na escuta              |
| QRR    | Apoio terrestre        |
| QRL    | Estou ocupado          |
| QRU    | Ocorrência             |
| QRV    | Às ordens              |
| QSJ    | Dinheiro               |
| QSM    | Repetir a mensagem     |
| QTA    | Sair do local/cancelar |
| QTC    | Mensagem               |
| QSL    | Compreendido           |
| PTR    | Patrulhamento          |
| QSV    | Viatura                |
| TKS    | Obrigado               |
| QTX    | Saindo de serviço      |
| QRX    | Aguarde/cessar rádio   |
| QRA    | Nome/patente           |
| QTO    | Banheiro               |
| QTI    | A caminho              |
| QTH    | Localização            |
